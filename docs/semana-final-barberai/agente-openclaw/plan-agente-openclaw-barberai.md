# Plan Agente OpenClaw — BarberAI

## Objetivo del agente

Documentar el flujo esperado para apoyar la preparación operativa de publicaciones y mensajes de BarberAI en WhatsApp, Facebook e Instagram, sin ejecutar contactos reales ni publicar sin revisión.

El agente debe servir como guía de trabajo para ordenar tareas repetitivas de campaña, registrar evidencia y preparar materiales bajo el tono Simplexity.

## Qué debe hacer

- Preparar mensajes de abordaje para BarberAI con tono profesional, claro, cercano y directo.
- Apoyar la preparación de publicaciones para Facebook e Instagram.
- Mantener el CTA principal orientado a WhatsApp.
- Registrar evidencia de cada paso cuando se realicen pruebas internas.
- Respetar la guía de marca Simplexity y los briefs aprobados.
- Detenerse cuando una acción requiera validación humana.

## Canales que debe manejar

- WhatsApp Web.
- Facebook.
- Instagram.

## Flujo esperado

1. Abrir WhatsApp.
2. Preparar mensaje de abordaje.
3. Navegar a Facebook/Instagram.
4. Publicar o preparar publicación.
5. Registrar evidencia.

## Qué NO debe hacer

- No contactar prospectos reales esta semana.
- No publicar sin revisión.
- No inventar respuestas.
- No guardar credenciales en archivos públicos.
- No afirmar que una publicación fue realizada si solo quedó preparada.
- No reportar validación de Allan sin evidencia.

## Estado

Documentado / Pendiente de configuración real.
