## Marca Simplexity

La documentacion y briefs usan el tono Simplexity: profesional, cercano, claro, directo y hondureno profesional.

Paleta oficial:

- Navy Blue: `#030065`
- Palatinate Blue: `#0038FF`
- Electric Blue: `#28EDFF`
- Mindaro: `#C6FF8D`
- UT Orange: `#FB8500` solo como acento puntual
- Blanco y negro para texto y fondos

Tipografia recomendada: Montserrat si no esta disponible Sofia Pro o Sofia Sans.

## Reglas de campania

- Cada pieza debe mantener un solo mensaje.
- Cada pieza debe mantener un solo CTA.
- El CTA principal debe ser WhatsApp.
- Las piezas deben mostrar barberias, barberos, clientes o escenas realistas del negocio.
- No usar disenos genericos solo con capturas del sistema.
