## Criterio de marca

La base usa el tono Simplexity: profesional, cercano, claro, directo y consistente. La producción visual debe respetar la paleta oficial:

- Navy Blue: `#030065`
- Palatinate Blue: `#0038FF`
- Electric Blue: `#28EDFF`
- Mindaro: `#C6FF8D`
- UT Orange: `#FB8500` solo como acento puntual
- Blanco y negro para texto y fondos
