# Checklist de configuración OpenClaw + Codex

Checklist para dejar preparado el agente comercial asistido de BarberAI antes de cualquier prueba real.

- [ ] Instalar o verificar OpenClaw.
- [ ] Verificar que Codex esté disponible como runtime.
- [ ] Crear perfil de navegador separado para el agente.
- [ ] No usar navegador personal.
- [ ] Verificar acceso a Meta Business Suite.
- [ ] Verificar página de Facebook.
- [ ] Verificar cuenta profesional de Instagram.
- [ ] Verificar WhatsApp Business o WhatsApp Business Platform.
- [ ] Crear mensajes base de WhatsApp.
- [ ] Crear captions base para Facebook.
- [ ] Crear captions base para Instagram.
- [ ] Crear sistema de aprobación manual.
- [ ] Crear reporte de acciones.
- [ ] Hacer prueba simulada sin publicar.
- [ ] Dejar evidencia para Allan.
