## Marca Simplexity

La documentación y briefs usan el tono Simplexity: profesional, cercano, claro, directo y hondureño profesional.

Paleta oficial:

- Navy Blue: `#030065`
- Palatinate Blue: `#0038FF`
- Electric Blue: `#28EDFF`
- Mindaro: `#C6FF8D`
- UT Orange: `#FB8500` solo como acento puntual
- Blanco y negro para texto y fondos

Tipografía recomendada: Montserrat si no está disponible Sofia Pro o Sofia Sans.

## Reglas de Campaña

- Cada pieza debe mantener un solo mensaje.
- Cada pieza debe mantener un solo CTA.
- El CTA principal debe ser WhatsApp.
- Las piezas deben mostrar barberías, barberos, clientes o escenas realistas del negocio.
- No usar diseños genéricos solo con capturas del sistema.
