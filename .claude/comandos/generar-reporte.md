# Generar Reporte

El agente genera un resumen de acciones comerciales preparadas para BarberAI.

El reporte debe incluir:

- Acciones preparadas.
- Acciones pendientes.
- Riesgos detectados.
- Siguiente paso recomendado.

El reporte debe aclarar si las acciones están solo preparadas, aprobadas o enviadas manualmente. No debe indicar que algo fue publicado o enviado si no existe confirmación humana.
