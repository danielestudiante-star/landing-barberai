## Criterio de marca

- Usar únicamente la paleta Simplexity: `#030065`, `#0038FF`, `#28EDFF`, `#C6FF8D`, blanco, negro y `#FB8500` solo como acento puntual.
- Usar Montserrat como tipografía principal si no está disponible Sofia Pro o Sofia Sans.
- Mantener tono profesional, cercano, claro y directo.
- Cada pieza debe tener un solo mensaje y un solo CTA.
- Las imágenes deben mostrar barberías o personas reales, o indicar claramente que se requiere foto realista de barbería/persona.
