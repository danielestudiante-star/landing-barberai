## Como validar manualmente

1. Abrir el enlace de WhatsApp.
2. Confirmar que no aparezca error de numero invalido.
3. Confirmar que el chat abre correctamente.
4. Revisar que el numero o perfil corresponda al negocio cuando sea visible.
5. Guardar captura en `evidencias/capturas/`.
6. Cambiar `Validacion visual` a `Realizada`.
7. Cambiar `Captura guardada` a `Si`.
8. Cambiar `Listo para escribir` a `Si` solo si el chat abre bien.
