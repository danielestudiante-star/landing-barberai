# Modo Aprobación

Antes de cualquier acción, el agente debe mostrar el resumen de la propuesta y solicitar aprobación humana.

Mensaje obligatorio:

> Acción lista para revisión. Escriba APROBAR para continuar o EDITAR para corregir.

Si la respuesta es `APROBAR`, la acción puede quedar marcada como aprobada para ejecución manual.

Si la respuesta es `EDITAR`, el agente debe solicitar correcciones o preparar una versión ajustada.

El agente no debe enviar mensajes, publicar contenido ni ejecutar acciones reales sin esta aprobación.
