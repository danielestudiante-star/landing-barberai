# Agente OpenClaw + Codex para apoyo comercial BarberAI

Esta entrega documenta y prepara un agente OpenClaw + Codex como asistente comercial para BarberAI. Su objetivo es apoyar el trabajo comercial de la próxima semana sin enviar mensajes, publicar contenido ni ejecutar acciones reales sin aprobación humana.

El agente servirá para ayudar a:

- Preparar mensajes de abordaje para clientes.
- Revisar prospectos antes de contactarlos.
- Preparar publicaciones para Facebook.
- Preparar publicaciones para Instagram.
- Organizar respuestas frecuentes.
- Dejar listo el flujo para la próxima semana.
- Mantener aprobación humana antes de enviar o publicar.

## Alcance

El alcance actual es documental y operativo inicial. No hay conexión real configurada con WhatsApp, Facebook ni Instagram desde este repositorio.

El agente debe trabajar como apoyo para Daniel, Allan o el responsable comercial, preparando textos y recomendaciones que luego serán revisadas manualmente.

## Límites obligatorios

- No crear un bot de spam.
- No automatizar mensajes masivos.
- No guardar contraseñas, tokens, cookies ni sesiones reales en el repositorio.
- No publicar contenido real automáticamente.
- No contactar prospectos sin canal verificado.
- No decir que existe conexión real con WhatsApp, Facebook o Instagram si solo está documentada o simulada.

## Archivos de esta entrega

- `flujo-agente-comercial.md`: flujo de entrada, análisis, acción propuesta, aprobación y salida.
- `politicas-seguridad.md`: reglas obligatorias de seguridad, privacidad y uso responsable.
- `checklist-configuracion.md`: pasos para dejar listo el entorno antes de pruebas reales.
- `mensajes-whatsapp.md`: mensajes base para abordaje y seguimiento comercial.
- `publicaciones-facebook-instagram.md`: publicaciones base para Facebook e Instagram.
- `reporte-estado.md`: reporte profesional para Allan.
